Краткая инструкция

Установка карт для Android:

Для установки карт достаточно скачать их через браузер на ваше устройство и 
поместить в каталог к другим картам.
То есть в каталог мапсми MapsWithMe в подкаталог с версией карт, например 191124.
Как автоматизировать данный процесс без платных менеджеров загрузок я не нашёл,
так что был бы признателен, если кому-то это удастся.


Установка карт для iOS:

Здесь файлы карт можно залить через iTunes, подключив ваше устройство к компьютеру.
Если через iTunes не получается, то можно попробовать через iFunbox.


Карты предназначены для последнего официального релиза MapsMe. Текущая версия 9.5.x.
